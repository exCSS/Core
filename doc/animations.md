#Animations

###How to use ***Animations***?

To use **Animations** just write it to the element (for example "fadeIn"):

ˋˋˋini

[#YourElement]
animation="fadeIn"

ˋˋˋ

###How to customize ***Animations***?

You can **customize** them with parameters:

ˋˋˋini

[#YourElement]
animation="fadeIn"
animation-parameter="delay:0s;duration:1.5s"

ˋˋˋ
